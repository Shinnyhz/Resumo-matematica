Resumo: Função Quadrática

1. Definição

Função do 2º grau: f(x) = ax² + bx + c, onde a ≠ 0.

2. Conceitos

  • Coeficiente a: define a concavidade (para cima se a > 0, para baixo se a < 0).
  • Coeficiente b: influencia a posição do vértice.
  • Coeficiente c: ponto onde a parábola cruza o eixo y.

3. Fórmula de Bhaskara

Δ = b² - 4ac

x = (-b ± √Δ) / 2a

4. Gráfico

  • Parábola com concavidade para cima ou para baixo.
  • Vértice: x = -b / 2a e y = f(x).
  • Interseções: Eixo y: (0, c), eixo x: raízes da equação.

5. Elementos da Parábola

  • Vértice
  • Eixo de simetria
  • Raízes (zeros)
  • Ponto y (c)
